# Live_project_1
Project Showcase: Bangalore Tech Salary Decoder

An intensive, single-sitting **2-hour data sprint** designed to build a functional salary-intelligence report analyzing the compensation structures of tech professionals in India's Silicon Valley.

#### ⚙️ Tech Stack & Tools
* **Language:** Python 🐍
* **Environment:** Google Colab / Jupyter Notebooks 📓
* **Libraries used:** Pandas, NumPy (Data Cleaning & Analytical Processing)

#### 📝 Project Overview & Implementation
* **The Goal:** Take raw data and transform it into an actionable, quantitative answer key broken down by role, experience, company type, skills, and education tier.
* **Dataset:** Cleaned and processed a dataset representing **1,000 Bengaluru tech professionals**.
* **Key Tasks:** 
  * Performed rigorous data cleaning to parse compensation metrics and handle empty values.
  * Conducted section-wise data analysis to extract median CTCs across roles like Product Managers, SDEs, and Data Scientists.
  * Generated a structured, production-ready notebook with section-wise commented code and a final printed report mapping out critical industry trends.
